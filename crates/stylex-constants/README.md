# `StyleX Constants`

## Overview

Static data and constant definitions for the StyleX SWC plugin.

## Contents

- `constants/common` -- CSS global values, valid callees, inject paths, compiled
  key markers
- `constants/messages` -- Error and warning message strings
- `constants/evaluation_errors` -- Evaluation-specific error messages
- `constants/priorities` -- CSS pseudo-class, pseudo-element, at-rule, and
  camelCase property priorities
- `constants/shorthands_of_longhands` -- CSS shorthand-to-longhand mappings
- `constants/shorthands_of_shorthands` -- CSS shorthand-to-shorthand mappings
- `constants/long_hand_logical` / `long_hand_physical` -- Logical and physical
  CSS longhand property sets
- `constants/logical_to_ltr` / `logical_to_rtl` -- Logical-to-physical property
  mappings for LTR/RTL
- `constants/length_units` / `time_units` -- CSS unit sets
- `constants/number_properties` / `unitless_number_properties` -- Numeric
  property metadata
- `constants/cursor_flip` -- Cursor value flipping for RTL
- `logger` -- `STYLEX_LOG_PREFIX` constant
