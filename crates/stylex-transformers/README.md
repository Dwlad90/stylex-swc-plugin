# `StyleX Transformers`

## Overview

This package provides the StyleX API transformer implementations for the SWC
plugin. Includes `stylex.create`, `stylex.defineVars`, `stylex.keyframes`,
`stylex.createTheme`, `stylex.firstThatWorks`, `stylex.types`,
`stylex.positionTry`, `stylex.viewTransitionClass` and `stylex.defineConsts`.
