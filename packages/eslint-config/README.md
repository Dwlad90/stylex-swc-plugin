# `@stylexswc/eslint-config`

> Part of the [StyleX SWC Plugin](https://github.com/Dwlad90/stylex-swc-plugin#readme) workspace

## Overview

Shared ESLint configuration preset for consistent code quality across all
TypeScript and JavaScript packages in the workspace.

## Development

```bash
pnpm --filter @stylexswc/eslint-config lint:check
```
